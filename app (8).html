Pulse
Mobile UX Concept

Омниканальный центр обработки обращений для модераторов Ассоциации частнопрактикующих психологов. Telegram, VK, Web — в одном интерфейсе с AI-ассистентом.

Скролл
01 — Информационная архитектура

Навигация и структура

Bottom Navigation с 4 вкладками — оптимальный паттерн для one-hand use. Каждая вкладка соответствует ключевому сценарию модератора.

Inbox

Главный экран. Сегменты: Мои / Нераспределённые / Все. Поиск, фильтры статуса, карточки чатов с превью и метаданными.

Active

Только активные диалоги текущего модератора. Быстрый доступ к незакрытым обращениям без лишнего шума.

AI Assistant

Суммаризация, предложение ответов, план решения. Human-in-the-loop: AI предлагает, модератор решает.

Settings

Профиль, уведомления, тема (light/dark), звук, управление шаблонами быстрых ответов.

04 — Компонентная система

Flutter + Vue Component Mapping

Двойная спецификация: Flutter-нативные виджеты и Vue + DaisyUI компоненты для Capacitor-сборки. Единая дизайн-система, два стека.

Flutter

AppShell

Scaffold + BottomNavigationBar с 4 вкладками. SafeArea для отступов. IndexedStack для сохранения состояния вкладок.

Vue + DaisyUI

AppShell

<btm-tabs> DaisyUI navbar внизу. safe-area-inset-* через CSS env(). <keep-alive> + <router-view>.

Flutter

InboxList

CustomScrollView + SliverAppBar (pinned search). SliverList с ChatCard виджетами. RefreshIndicator для pull-to-refresh.

Vue + DaisyUI

InboxList

<InboxList> с v-for по чатам. DaisyUI input для поиска, tabs для сегментов. Pull-to-refresh через Capacitor plugin.

Flutter

ChatThread

ListView.builder reverse для сообщений. MessageBubble виджет. Sticky ComposerBar через bottomSheet в Scaffold.

Vue + DaisyUI

ChatThread

<MessageBubble> компонент. Scroll container с overflow-y: auto. Fixed <ComposerBar> с учётом keyboard overlap.

Flutter

AI Assistant

showModalBottomSheet с DraggableScrollableSheet. Карточки AI-ответов как Card виджеты. Shimmer для loading.

Vue + DaisyUI

AiAssistPanel

DaisyUI drawer или modal bottom-sheet. <AiCard> компоненты. Skeleton loader через Tailwind animate-pulse.

Flutter

ComposerBar

TextField + Row с иконками. Wrap для quick chips. AnimatedContainer для плавного расширения.

Vue + DaisyUI

ComposerBar

DaisyUI textarea + button. Chips как badge с @click. env(safe-area-inset-bottom) для padding.

Состояния UI

Loading

Skeleton-карточки в списке. Shimmer-эффект в чате. Пульсирующий индикатор AI-обработки.

Empty State

Иллюстрация + текст «Нет обращений». Разные для каждого сегмента. CTA для обновления.

No Network

Баннер вверху экрана. Кэшированные данные доступны. Retry-кнопка. Toast при восстановлении.

Error

Inline-ошибка с retry. Не блокирует весь экран. Логирование для отладки. Понятный текст.

UX Rationale

Скорость модерации

3 тапа от пуша до ответа. Quick chips, AI-подсказки и шаблоны сокращают время ответа на 40-60%.

Снижение когнитивной нагрузки

Чёткая визуальная иерархия, цветовое кодирование каналов, AI triage для мгновенного понимания сути обращения.

One-hand use

Bottom navigation, composer внизу, swipe-действия. Все ключевые зоны в нижней трети экрана.

Эмпатичный тон

Спокойная палитра, мягкие скругления, отсутствие агрессивных красных. Профессионализм и поддержка в каждом пикселе.
